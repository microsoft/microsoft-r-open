About Microsoft R Open
-----------------------
Microsoft R Open builds on R {VERSION_NUMBER} by making the following additions and changes:

The default CRAN repository for this version of R Open can be found at http://mran.microsoft.com/snapshot/.

Unlike a CRAN mirror, this is a static copy of CRAN. Newer versions of packages will not appear in this repository. To access newer versions of packages, choose a CRAN mirror or use the checkpoint package described below.

Microsoft R Open installs the `checkpoint` package. You can use the checkpoint package to access recent package versions and to share reproducible R code with others. See https://mran.microsoft.com/package/checkpoint for information about how to use the checkpoint package to manage R package versions.
